🚀 Portfolio de Desarrollo

Este repositorio contiene mi portfolio personal, donde iré publicando todos mis proyectos, organizados desde los más recientes hasta los primeros que desarrollé.


El objetivo de este portfolio es mostrar mi evolución técnica, mi forma de estructurar proyectos y mi enfoque en la calidad del código.


🛠️ Stack Tecnológico

El frontend del portfolio está desarrollado con:

React

JavaScript

CSS

La mayoría de los proyectos estarán desplegados en Vercel, incluyendo sus respectivas bases de datos cuando sea necesario.

Algunos proyectos aún se encuentran en desarrollo, y otros serán añadidos progresivamente.


📐 Filosofía de Desarrollo

Todos los proyectos incluidos en el portfolio seguirán los siguientes principios:

✅ Diseño responsive

✅ Código limpio y comentado

✅ Funciones pequeñas y legibles

✅ Estructura modular y escalable

✅ Separación clara de responsabilidades

✅ Buenas prácticas de desarrollo

El objetivo es mantener una arquitectura clara que facilite el mantenimiento, la escalabilidad y la comprensión del código.
